# 어휘 

의사소통능력 하위 어휘를 기록하기 위한 노트 

## 서다

| 표현 | 의미 | 예문 |
|------|------|------|
| **줄을 서다** | 차례를 기다리며 늘어서다 | 창구 앞에 줄을 섰다 |
| **앞에 서다** | 선두·책임 위치에 나서다 | 팀장이 앞에 섰다 |
| **편에 서다** | 어느 쪽을 지지하다 | 동생 편에 서서 말했다 |
| **체면이 서다** | 남 앞에서 체면이 유지되다 | 그 말 한마디에 체면이 섰다 |
| **결심이 서다** | 마음이 굳게 정해지다 | 드디어 결심이 섰다 |
| **논리가 서다** | 주장의 근거가 타당하다 | 설명을 들으니 논리가 섰다 |
| **날이 서다** | 칼·말 등이 예리해지다 | 말에 날이 서 있다 |
| **소름이 서다** | 소름이 돋다 | 무서워서 소름이 섰다 |
| **아이가 서다** | 뱃속에 아이가 생기다 (임신하다) | 아이가 선 것 같다 |
