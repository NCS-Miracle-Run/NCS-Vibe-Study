## 정보능력 소개 

### 자료-정보-지식의 차이 

* 자료-정보-지식-정보처리의 의미 

* 자료
    * 단순한 사실의 나열이자 정보 작성을 위해 필요한 데이터
    * 객관적 실제의 반영이며, 그것을 전달할 수 있도록 기호화한 것
    * 아직 특정한 목적에 대하여 평가되지 않은 상태의 숫자나 문자들의 단순한 나열

* 정보
    * 의미있는 자료로, 자료를 특정한 목적과 문제해결에 도움이 되도록 가공한 것
    * 자료를 일정한 프로그램에 따라 컴퓨터가 처리/가공함으로써 특정한 목적을 달성하는데 필요한 것. 특정한 의미를 지닌 것으로 다시 생산한 것

* 지식
    * 가치 있는 정보이며, 특정한 목적을 달성하기 위해 과학적 또는 이론적으로 추상화되거나 정립되어 있는 일반화된 정보
    * 어떤 대상에 대하여 원리적/통일적으로 조직되어 객관적 타당성을 요구할 수 있는 판단의 체계를 제시함

* 정보처리
    * 자료를 가공하여 이용 가능한 정보로 만드는 과정
    * 자료처리라고도 하며 일반적으로 컴퓨터가 담당함

### 정보의 특성

* 적시성과 독점성은 정보의 핵심적인 특성으로, 공개 정보보다는 반공개 정보가, 반공개 정보보다는 비공개 정보가 더 큰 가치를 가질 수 있음
* 공개 정보와 비공개 정보를 적절히 구분함으로써 경제성과 경쟁성을 동시에 추구해야 함 

### 정보의 가치

켄트로의 지식 삼각형: 다음 순으로 가치가 높음

1. 지혜: 지식을 활용하는 창의적 아이디어
2. 지식: 정보들 간의 관계를 통해 얻는 가치 있는 정보
3. 정보: 유의미하게 가공한 2차 자료
4. 데이터: 가공하기 전 순수한 상태의 수치

* 정보의 가치: 우리의 요구, 사용 목적, 활용 시기 및 장소에 따라서 다르게 평가됨 

### 정보화 사회

* 정보화 사회의 의미 
    * 이 세상에서 필요로 하는 정보가 중심이 되는 사회 
    * 컴퓨터 기술과 정보통신 기술을 활용하여 사회 각 분야에서 필요로 하는 가치 있는 정보를 창출하고, 보다 유익하고 윤택한 생활을 영위하는 사회로 발전시켜 나가는 것 

* 미래의 사회 
    1. 부가가치 창출 요인이 토지, 자본, 노동에서 지식 및 정보 생산 요소로 전환 
        * 지식/정보가 부가가치 창출의 3/4를 차지함 
        * 정보기술(IT) 이후 차세대 대표 주력 산업은 생명공학(BT), 나노공학(NT), 환경공학(ET), 문화산업(CT), 우주항공기술(ST)의 6T로 나타냄
    2. 세계화의 진전 
        * 모든 국가의 시장이 국경 없는 하나의 세계 시장으로 통합됨 
        * 실물 상품뿐만 아니라 노동, 자본, 기술 등의 생산 요소와 교육과 같은 서비스의 국제 교류도 모두 포함함 
    3. 지식의 폭발적인 증가 
        * 특히 과학적 지식이 폭발적으로 증가함 

* 정보화 사회에서 해야 할 일: 정보검색, 정보관리, 정보전파

### 컴퓨터 활용 분야

* 기업 경영 분야: 생산에서부터 판매, 회계, 재무, 인사 및 조직 관리, 금융 업무 등에 널리 활용
    * 경영정보시스템(MIS): Management Information System. 기업 경영정보를 총괄해 의사결정을 지원하는 시스템
    * 의사결정지원시스템(DSS): Decision Support System. 기업경영에 필요한 정보 활용을 지원해 경영자가 신속한 의사결정을 할 수 있도록 돕는 시스템
    * 사무자동화(OA): Office Automation. 문서 작성과 보관은 물론 컴퓨터로 업무를 결재하는 전자 결재 시스템 도입
    * 전자상거래(EC): Electronic Commerce. 생산에서 소비까지 전 과정을 컴퓨터로 처리. B2B(기업 간 거래), B2C(인터넷 소매법), B2G(기업과 정부 간 전자상거래)

* 행정분야: 민원 처리, 각종 행정 통계 등 여러 행정 관련 정보를 데이터베이스로 구축

* 산업분야: 공업, 상업 등 각 분야에서 널리 활용될 뿐만 아니라 중요한 역할을 담당함
    * 공정자동화: Factory Automation. 컴퓨터를 이용하여 제품의 수주에서부터 설계, 제조, 검사, 출하에 이르기까지의 모든 제품 공정 과정을 자동화
    * 판매시점관리: POS. 편의점이나 백화점에서의 매출액 계산, 원가 및 재고 관리 등에 컴퓨터 활용
    * 전자문서교환: EDI. 전자상거래에서 데이터를 효율적으로 교환하기 위한 데이터와 문서의 표준화 시스템

* 기타분야: 교육, 연구소, 출판, 가정, 도서관, 예술 분야 등에 널리 활용됨

### IT 기기를 활용한 정보 처리 과정 

* 순서: 정보의 기획 > 정보의 수집 > 정보의 관리 > 정보의 활용 

* 정보의 기획 
    * 5W 2H 기획 
    * What: 입수 대상
    * Where: 정보의 소스 
    * When: 정보의 요구 
    * Why: 정보의 필요목적 
    * Who: 정보활동의 주체 
    * How: 정보의 수집 방법 
    * How Much: 정보 수집의 비용

* 정보의 수집
    * 다양한 정보원으로부터 목적에 적합한 정보를 입수하는 것 
    * 정보 수집의 최종적인 목적은 '예측'을 잘하는 것 

* 정보의 관리 
    * 수집된 다양한 형태의 정보를 문제해결이나 결론 도출에 사용하기 쉬운 형태로 바꾸는 일 
    * 정보 관리의 3원칙 
    * 목적성: 사용 목적을 명확히 설명할 것 
    * 용이성: 쉽게 작업할 수 있을 것 
    * 유용성: 즉시 사용할 수 있을 것 

* 정보의 활용 
    * 정보기기에 대한 이해나 최신 정보기술이 제공하는 주요 기능과 특성에 대한 지식을 아는 능력이 필요함 
    * 지식을 아는 것 외에도 정보가 필요하다는 문제 상황을 인지하는 능력, 문제해결에 적합한 정보를 찾고 선택하는 능력, 찾은 정보를 문제해결에 적용하는 능력, 윤리의식을 가지고 합법적으로 정보를 활용하는 능력 등 다양한 능력이 필요함 


## 컴퓨터활용능력 

### 다양한 인터넷 서비스 

* 이메일: 인터넷을 통해 편지나 정보를 주고받는 서비스 

* 메신저: 인터넷에서 실시간으로 메시지와 데이터를 주고받을 수 있는 소프트웨어 

* 인터넷 디스크/웹하드 
    * 웹 서버의 대용량 저장 기능을 갖추고 사용자가 개인용 컴퓨터의 하드디스크와 같은 기능을 인터넷을 통하여 이용할 수 있게 하는 서비스 
    * 인터넷 디스크, 웹 디스크, 웹하드, 파일 박스, 피디 박스 등은 모두 같은 서비스 

* 클라우드 
    * 별도의 데이터 센터 없이 인터넷을 통해 제공되는 서버를 활용해 정보를 보관하고 필요할 때 꺼내 쓰는 기술 
    * 작업한 컴퓨터에서만 아니라 언제 어디서나 필요한 자료를 불러올 수 있음 

* SNS 
    * 온라인 인맥 구축을 목적으로 개설된 커뮤니티형 웹사이트 
    * 현재 다른 사람과 의사소통을 하거나 정보를 공유/검색하는데 일상적으로 이용 됨 

* 전자상거래 
    * 인터넷이라는 전자매체를 통하여 상품을 사고팔거나, 재화나 용역을 거래하는 사이버 비즈니스 
    * 소비자와의 거래뿐만 아니라 공급자, 금융기관, 정부기관, 운송기관 등과 같은 거래에 관련되는 모든 기관과의 관련 행위 

### 인터넷을 이용한 정보 검색 

* 정보검색의 의미: 여러 곳에 분산되어 있는 수많은 정보 중에서 특정 목적에 적합한 정보만을 신속하고 정확하게 찾아내어 수집/분류/축적 하는 과정 

* 정보검색 단계: 검색 주제 선정 > 정보원 선택 > 검색식 작성 > 결과 출력 

* 검색엔진 유형 
    1. 키워드 검색방식
        * 찾고자 하는 정보와 관련된 핵심 언어인 키워드를 직접 입력해 검색엔진에 보내면 검색엔진이 키워드와 관련된 정보를 찾는 방식 
        * 키워드만 입력하면 되기 때문에 검색이 간단하나 키워드가 불명확하게 입력될 경우 검색 결과가 지나치게 많아 비효율적인 검색이 될 수 있음 
    2. 주제별 검색방식 
        * 인터넷상에 존재하는 웹 문서들을 주제/계층별로 정리하여 데이터베이스를 구축한 후 이용하는 방식 
        * 원하는 정보를 찾을 때까지 상위 주제부터 하위 주제까지 분류되어 있는 내용을 선택하여 검색하면 원하는 정보를 발견할 수 있음 
    3. 자연어 검색방식 
        * 검색엔진에서 문장형 질의어의 형태소를 분석해 5W 2H를 읽어내고 각 질문에 답이 들어 있는 사이트를 연결해주는 방식
    4. 통합형 검색방식
        * 사용자가 입력한 검색어들을 연계된 다른 검색엔진에 보내고, 이를 통해 얻어진 검색 결과를 사용자에게 보여주는 방식 

* 인터넷 정보검색을 할 때의 유의사항 
    * 키워드의 선택 
        * 키워드가 너무 짧으면 원하는 결과를 쉽게 찾을 수 없으므로 키워드는 구체적이고 자세하게 만들어야 함
        * 특정한 키워드에 대하여 검색 결과가 너무 많을 경우 검색엔진에서 결과 내 재검색 기능을 활용해 검색 결과의 범위를 좁힘 
    * 다양한 방법 활용 
        * 웹 검색 이외에 각종 전자게시판(BBS)이나 뉴스 그룹, 메일링 리스트를 이용하고, 도서관 자료와 정보를 가지고 있는 사람에게 직접 전자우편으로 부탁하는 등의 다른 방법도 적극 활용 
    * 검색엔진의 가중치에 대한 지나친 신뢰 주의 
        * 검색엔진 나름대로 정확성이 높다고 판단한 데이터를 화면 상단에 표시하나 실제로 그렇지 않은 경우가 많으므로 사용자 스스로 검색한 자료가 자신이 원하는 자료인지 판단해야 함 


### 업무에 필요한 소프트웨어 활용 

* 워드 프로세서 
    * 여러 형태의 문서를 작성/편집/저장/인쇄할 수 있는 프로그램 
    * 입력/표시/저장/편집/인쇄 기능 등 

* 스프레드시트 
    * 문서 작성 및 편집 기능 외에 수치나 공식을 입력해 그 값을 계산하고 계산 결과를 차트로 표시할 수 있는 전자계산표 또는 표 계산 프로그램 
    * 계산/수치/차트/저장/편집/인쇄 기능, 데이터베이스 및 그래픽 기능, 통신 기능 등

* 프레젠테이션 
    * 프레젠테이션 보고(보고/회의/상담/교육 등)에서 정보를 전달하는데 널리 활용되는 프로그램 
    * 저장/편집/인쇄/슬라이드 쇼 기능 등 

* 데이터베이스 
    * 대량의 자료를 관리하고 구조화하여 검색이나 자료관리 작업을 효과적으로 실행하는 프로그램 

* 그래픽 소프트웨어 
    * 새로운 그림을 그리거나 그림 또는 사진 파일을 불러와 편집하는 프로그램 

* 유틸리티 프로그램 
    * 사용자가 컴퓨터를 좀 더 쉽게 사용할 수 있도록 도와주는 소프트웨어 

* 소프트웨어 구분 
    * 시스템 소프트웨어: 하드웨어를 제어하고 응용 소프트웨어를 위한 기반 환경 제공 
    * 운용 소프트웨어: 운영 체제에서 실행되는 모든 소프트웨어 

### 데이터베이스 구축의 필요성

* 데이터베이스의 의미 
    * 데이터베이스: 서로 연관된 여러 개의 파일 
    * 데이터베이스 관리시스템: 데이터와 파일, 그들의 관계 등을 생성/유지/검색할 수 있게 해주는 소프트웨어 
    * 파일관리 시스템: 한 번에 한개의 파일에 대해서 생성/유지/검색할 수 있는 소프트웨어 

* 데이터베이스 구축의 필요성 
    1. 데이터의 중복 감소: 데이터를 한곳에서만 가지고 있으므로 데이터 유지 비용이 감소함
    2. 데이터 무결성 향상: 결함 없는 데이터를 유지하는 것이 용이하며, 데이터 변경 시 모든 애플리케이션을 최신화하는 것이 가능함 
    3. 검색 용이화: 원하는 검색이나 보고서 작성 등을 쉽게 할 수 있음 
    4. 데이터의 안정성 향상: 사용자가 정보에 대한 보안 등급을 정해 보안 유지가 가능함
    5. 프로그램의 개발 기간 단축: 프로그램의 개발이 쉬워지고 기간도 단축됨 

* 데이터베이스의 기능 
    * 입력 기능: 형식화된 폼을 사용하여 내용을 편리하게 입력 가능 
    * 데이터 검색 기능: 필터나 쿼리 기능을 이용하여 데이터의 빠른 검색 및 추출 가능 
    * 데이터 일괄 관리: 테이블을 사용하여 데이터 관리가 쉬우며, 많은 데이터를 종류별로 분류하여 일괄적으로 관리 가능 
    * 보고서 기능: 데이터베이스에 있는 데이터로 청구서나 명세서 등의 서류 작성을 쉽게할 수 있음 

* 데이터베이스의 작업 순서: 시작 > 데이터 베이스 만들기 > 자료 입력 > 저장 > 자료검색 > 보고서 인쇄 > 종료

## 정보처리능력 

### 정보의 수집 및 분석과 가공 

* 정보의 필요성: 의사결정을 하거나 문제의 답을 알아내고자 할 때 새로운 정보가 필요함 

* 정보원: 필요한 정보를 수집할 수 있는 원천으로, 공개된 것과 비공개된 것을 모두 포함함 
    * 1차 자료: 원래의 연구 성과가 기록된 자료. 단행본, 학술지와 학술지 논문, 학술회의자료, 연구보고서, 학위논문, 특허 정보, 표준 및 규격자료, 레터, 출판 전 배포자료, 신문, 전자, 웹 정보자원 등 
    * 2차 자료: 1차 자료를 효과적으로 찾아보기 위한 자료로, 1차 자료에 포함되어 있는 정보를 압축/정리하여 읽기 쉬운 형태로 제공하는 자료. 사전, 백과사전, 편람, 연감, 서지 데이터베이스 

* 정보수집 방법
    * 인포메이션 vs 인텔리전스: 단순한 인포메이션을 수집할 것이 아니라 직접적으로 도움을 줄 수 있는 인텔리전스를 수집할 필요가 있음 
        * 인포메이션: 하나하나의 개별적인 정보 
        * 인텔리전스: 수많은 인포메이션 중 몇 가지를 선별해 그것을 연결시켜 판단하기 쉽게 도와주는 하나의 정보 덩어리 
    * 선수필승: 다른 사람보다 1초라도 빨리 정보를 잡은 사람이 우위에 서게 됨 
    * 머릿속에 서랍 만들기: 머릿속에 서랍을 만들어 자기 나름대로 정리해 놓으면 효과적인 정보를 수집할 수 있음 
    * 정보수집용 하드웨어 활용: 물리적인 하드웨어를 활용해 기억력의 한계를 극복하고 나중에 유용해질 정보를 수집할 수 있음 

* 정보분석의 정의 
    * 여러 정보를 상호 관련지어 새로운 정보를 생성해내는 활동 
    * 한 개의 정보만으로는 불분명한 사항을 다른 정보를 통해 명백히 할 수 있으며, 서로 상반되거나 큰 차이가 있는 정보의 내용을 판단하여 새로운 해석도 가능 

* 정보 분석의 절차: 분석 과제의 발생 > 과제(요구)의 분석 > 조사 항목의 선정 > 관련 정보의 수집(기존자료 조사, 신규자료 조사) > 수집정보의 분류 > 항목별 분류 > 종합/결론 > 활용/정리 

* 정보분석의 중요성 
    * 좋은 정보의 수집과 훌륭한 분석은 업무수행에서 매우 중요함 
    * 훌륭한 분석이란 하나의 메커니즘을 그려낼 수 있고, 동향과 미래를 예측할 수 있는 것 
    * 1차 정보를 분석/압축/가공하여 2차 정보 작성 
        * 1차 정보 분석/압축/가공: 1차 정보가 포함하는 내용을 몇개의 설정된 카테고리로 분석하여 각 카테고리의 상관관계를 확장함 
        * 2차 정보 작성: 1차 정보가 포함하는 주요 개념을 대표하는 용어를 추출하며, 이를 간결하게 서열화/구조화함 

### 정보의 효율적인 관리 및 효과적인 활용 

* 정보관리의 방법과 특징 
    1. 목록을 이용한 정보 관리 
        * 정보 중에서 중요한 항목을 찾아 기술한 후 정리하면서 만듦 
        * 목록을 디지털 파일로 저장해 놓으면 특정 용어를 이용하여 검색 가능 
    2. 색인을 이용한 정보 관리 
        * 주요 키워드나 주제어를 가지고 소장하고 있는 정보(Source)를 관리하는 방식 
        * 색인은 정보를 찾을 때 사용 할 수 있는 키워드인 색인어와 색인어의 출처인 위치정보로 구성됨 
        * 디지털 파일에 색인을 저장할 경우 추가/삭제/변경이 용이함 
    3. 분류를 이용한 정보관리: 유사한 정보끼리 모아 체계화하여 정리 
        * 시간적 기준: 정보의 발생 시간별로 분류 
        * 주제적 기준: 정보의 내용에 따라 분류
        * 기능적/용도별 기준: 정보가 이용되는 기능 및 용도에 따라 분류 
        * 유형적 기준: 정보의 유형에 따라 분류 


* 정보 활용 형태 
    * 수집한 정보를 그대로 활용 
    * 수집한 정보를 그대로 활용하되 일정한 형태로 표현하여 활용 
    * 수집한 정보를 정리/분석/가공하여 활용 
    * 수집한 정보를 정리/가공하여 활용하되 일정한 형태로 표현하여 활용 
    * 생산된 정보를 일정한 형태로 재표현하여 활용 
    * 일정한 형태로 표현한 정보, 한 번 이용한 정보를 보존 및 정리하여 장래에 활용 

* 동적 정보
    * 신문, TV 뉴스 등 시시각각으로 변화하는 정보 
    * 대부분 밀려와 쌓이기만 하는 정보로 미련 없이 버려도 무방함 
    * 시의적절하게 수집/관리하고 활용하는 것이 중요함 

* 정적 정보 
    * 잡지나 책에 있는 정보, CD-ROM이나 USB에 수록된 영상정보 등 보존되어 멈추어 있는 정보 
    * 생각하고 보존함 

### 사이버 공간에서의 예절 및 개인정보 유출 방지법

* 네티켓
    * 네트워크와 예절을 뜻하는 에티켓의 합성어로, 네티즌이 사이버 공간에서 지켜야 할 비공식적인 규약 
    * 법적 제재에 의존하는 타율적 해결보다 네티즌이 자율적으로 사이버 공간의 문제를 미리 방지하고 이성적으로 해결해 나가자는 적극적 의미 

* 다양한 네티켓 
    * E-mail
        * 메시지는 가능한 짧게 요점만 작성할 것 
        * 메일을 보내기 전에 주소가 올바른지 다시 한번 확인할 것 
        * 제목은 메시지 내용을 함축해 간략하게 작성할 것 
        * 가능하면 메시지 끝에 Signature(성명, 직위, 단체명, 메일주소, 전화번호 등)를 포함하되, 너무 길지 않도록 작성할 것 
        * 메일상에서 타인에 대해 말할 때는 정중함을 지키고, 비방이나 욕설같이 타인에게 피해를 주는 언어는 사용하지 않을 것 
    * 채팅 
        * 마주 보고 이야기하는 마음가짐으로 임할 것 
        * 대화방에 들어가면 지금까지 진행된 대화의 내용과 분위기를 확인할 것 
        * Enter를 누르기 전 한번 더 생각할 것 
        * 광고나 홍보 등을 목적으로 악용하지 말 것
        * 유언비어, 속어와 욕설 게재는 삼가고, 상호 비방의 내용은 금할 것
    * 게시판 
        * 글의 내용은 간결하게 요점만 작성할 것 
        * 제목에는 글의 내용을 파악할 수 있는 함축된 단어를 쓸 것 
        * 글의 내용 중에 잘못된 점이 있으면 빨리 수정하거나 삭제할 것 
        * 타인의 의견에 대해 무조건적인 비판 및 비방을 하거나 유언비어를 남기지 말 것 
        * 게시판의 주제와 관련 없는 내용은 올리지 않을 것 
    * 공개 자료실
        * 음란물을 올리지 않을 것 
        * 상업용 소프트웨어를 올리지 않을 것 
        * 공개 자료실에 등록된 자료는 가급적 압축할 것 
        * 프로그램을 올릴 때는 사전에 바이러스 감염 여부를 점검할 것 
        * 유익한 자료를 받았을 때는 올린 사람에게 감사의 편지를 보낼 것 
    * 인터넷 게임 
        * 상대방을 존중하며, 경어를 사용할 것 
        * 인터넷 게임에 너무 집착하지 말 것 
        * 온라인 게임은 온라인상의 오락으로 끝낼 것 
        * 게임 중에 일방적으로 퇴장하는 것은 무례한 일임을 명시할 것 
        * 게이머도 일종의 스포츠맨이므로 스포츠맨십을 가질 것 
        * 이겼을 때는 상대를 위로하고 졌을 때는 깨끗하게 물러설 것 

* 인터넷의 역기능과 유의 사항 
    * 불건전한 정보의 유통: 음란 사이트, 엽기 사이트, 도박 사이트, 폭력 사이트, 반사회적 사이트 등 
    * 컴퓨터 바이러스: 컴퓨터 내부에 침투하여 자료를 손상시키거나 다른 프로그램을 파괴시키는 컴퓨터 프로그램의 일종 
    * 사이버 언어폭력: 욕설, 비방, 도배, 성적 욕설, 유언비어, 악성 댓글 등 
    * 사이버 성폭력: 인터넷 채팅이나 게시판, SNS 등을 통해 성적으로 수치심을 주는 행위 
    * 인터넷 중독: 인터넷 이용이 보편화되면서 인터넷에 지나치게 빠져 생활이 곤란한 경우 
    * 저작권 침해: 불법으로 복제된 소프트웨어 파일 등을 배포하거나 저작권자의 동의 없이 공개하는 행위 
    * 해킹: 다른 시스템에 불법으로 침입하여 시스템에 저장된 정보를 임의로 변경, 삭제 또는 절취하는 행위
    * 개인정보 유출: 해킹이나 바이러스 감염 등으로 개인정보가 누출되어 사생활을 침해받는 경우 

* 개인 정보
    * 생존하는 개인에 관한 정보로서, 개인을 식별할 수 있는 이름, 주민등록번호 등의 정보 
    * 해당 정보만으로는 특정 개인을 식별할 수 없더라도 다른 정보와 용이하게 결합하여 식별할 수 있는 것도 개인정보에 포함됨 

* 개인정보의 종류: 일반 정보, 가족 정보, 교육 및 훈련 정보, 병역 정보, 부동산 및 동산 정보, 소득 정보, 기타 수익, 신용 정보, 고용 정보, 법적 정보, 의료 정보, 조직 정보 

* 개인정보 유출 방지 방법 
    * 회원 가입 시 이용 약관을 읽을 것 
    * 이용목적에 부합하는 정보를 요구하는지 확인할 것 
    * 비밀번호는 정기적으로 교체할 것 
    * 정체불명의 사이트는 멀리할 것 
    * 가입 해지 시 정보 파기 여부를 확인할 것 
    * 뻔한 비밀번호를 쓰지 말 것 

* 컴퓨터 바이러스 예방법 
    * 출처가 불분명한 전자우편의 첨부파일은 백신 프로그램으로 바이러스 검사 후 사용 
    * 실시간 감시 기능이 있는 백신 프로그램을 설치하고 정기적으로 업데이트 
    * 바이러스가 활동하는 날에는 시스템을 사전에 검사 
    * 정품 소프트웨어를 구입하며 사용하는 습관 
    * 중요한 파일은 습관적으로 별도의 보조 기억 장치에 미리 백업 
    * 프로그램을 복사할 때는 바이러스 감염 여부 확인

    * Alt + Enter: 선택한 파일 또는 폴더의 속성 대화상자 열기 
    * Alt + Esc: 실행 중인 프로그램 창을 순서대로 전환 
    * Alt + Tab: 활성화되어 있는 프로그램 창 전환

* microsoft office excel 기본 함수식
    * IF: 조건이 참이면 값1, 거짓이면 값2 반환 → `=IF(조건, 값1, 값2)`
    * COUNT: 숫자가 있는 셀 개수 → `=COUNT(범위)`
    * COUNTA: 비어있지 않은 셀 개수(문자 포함) → `=COUNTA(범위)`
    * COUNTIF: 단일 조건을 만족하는 셀 개수 → `=COUNTIF(범위, 조건)`
    * COUNTIFS: 여러 조건을 모두 만족하는 셀 개수 → `=COUNTIFS(범위1, 조건1, 범위2, 조건2, ...)`
    * SUM: 합계 → `=SUM(범위)`
    * SUMIF: 단일 조건을 만족하는 셀들의 합계 → `=SUMIF(조건범위, 조건, 합계범위)`
    * AVERAGE: 평균(숫자만) → `=AVERAGE(범위)`
    * AVERAGEA: 평균(문자는 0으로 처리) → `=AVERAGEA(범위)`
    * AVERAGEIF: 단일 조건을 만족하는 셀들의 평균 → `=AVERAGEIF(조건범위, 조건, 평균범위)`
    * ROUND: 지정한 자릿수로 반올림 → `=ROUND(수, 자릿수)` (양수=소수점 아래, 음수=정수 자리)
    * MAX: 최댓값 → `=MAX(범위)`
    * MIN: 최솟값 → `=MIN(범위)`
    * LARGE: k번째로 큰 값 → `=LARGE(범위, k)`
    * SMALL: k번째로 작은 값 → `=SMALL(범위, k)`
    * REPLACE: 문자열의 특정 위치부터 n개 문자를 다른 문자로 교체 → `=REPLACE(원본, 시작위치, 문자수, 새문자)`
    * VLOOKUP: 열 기준 세로 방향 검색 → `=VLOOKUP(찾을값, 표범위, 열번호, [유사일치])` (유사일치: TRUE=근사값, FALSE=정확히)
    * HLOOKUP: 행 기준 가로 방향 검색 → `=HLOOKUP(찾을값, 표범위, 행번호, [유사일치])`
    * RIGHT: 오른쪽에서 n개 문자 추출 → `=RIGHT(문자열, n)`
    * MID: 지정 위치부터 n개 문자 추출 → `=MID(문자열, 시작위치, n)`
    * LEFT: 왼쪽에서 n개 문자 추출 → `=LEFT(문자열, n)`
    * RANK: 순위 반환 → `=RANK(값, 범위, [순서])` (순서: 0=내림차순, 1=오름차순)
    * INDEX: 범위에서 지정한 행·열 위치의 값 반환 → `=INDEX(범위, 행번호, 열번호)`
    * DSUM: 데이터베이스에서 조건을 만족하는 필드의 합계 → `=DSUM(데이터베이스, 필드, 조건범위)`
    * DCOUNT: 데이터베이스에서 조건을 만족하는 숫자 셀 개수 → `=DCOUNT(데이터베이스, 필드, 조건범위)`
    * CORREL: 두 데이터 범위의 상관계수(-1~1) 반환 → `=CORREL(범위1, 범위2)`
