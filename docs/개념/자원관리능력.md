## 1. 자원의 개념 및 관리 과정

### 자원의 개념

자원이란 직업생활에서 목표를 달성하기 위해 활용되는 **시간, 돈(예산), 물적자원, 인적자원** 등을 의미한다.

| 자원 | 설명 |
|------|------|
| 시간 | 업무 수행에 사용 가능한 시간 |
| 예산 | 업무에 투입되는 비용·자금 |
| 물적자원 | 장비, 시설, 재료 등 물리적 자원 |
| 인적자원 | 업무를 수행하는 사람(인력, 역량) |

> 자원은 **유한하다** — 낭비 없이 효율적으로 활용해야 한다.

---

### 자원의 낭비요인

자원이 효과적으로 활용되지 못하는 주요 원인 4가지:

| 낭비요인 | 설명 |
|----------|------|
| **비계획적 행동** | 계획 없이 즉흥적으로 행동하여 자원을 낭비 |
| **편리성 추구** | 당장 편한 방법을 선택해 비효율 발생 |
| **자원에 대한 인식 부재** | 자원의 유한성을 인식하지 못해 무분별하게 사용 |
| **노하우 부족** | 효율적인 자원 활용 방법을 몰라 낭비 발생 |

---

### 효과적인 자원관리 과정

효과적인 자원관리는 아래 4단계 순서로 이루어진다:

```
① 필요한 자원 확인 → ② 이용 가능한 자원 수집 → ③ 자원 활용 계획 수립 → ④ 계획에 따른 수행
```

| 단계 | 내용 |
|------|------|
| **1단계: 필요한 자원 확인** | 업무 목표 달성에 어떤 자원이 얼마나 필요한지 파악 |
| **2단계: 이용 가능한 자원 수집** | 실제로 확보 가능한 자원의 종류·양을 조사 |
| **3단계: 자원 활용 계획 수립** | 수집된 자원을 어떻게 배분·사용할지 계획 작성 |
| **4단계: 계획에 따른 수행** | 계획대로 자원을 투입하고 실행 |

> **핵심 포인트:** 자원관리는 무작정 실행하는 것이 아니라, **계획 수립 후 수행**하는 것이 원칙이다.

## 2. 시간자원관리

### 시간의 특성

- 매일 24시간
- 속도가 일정
- 멈출 수 없음
- 빌리거나 저축할 수 없음
- 사용 방법에 따라 가치가 다름
- 시기에 따라 밀도가 다름

---

### 시간 관리의 효과

**개인 관점**
- 생산성 향상
- 스트레스 감소
- 균형적인 삶 
- 목표 성취

**기업 관점**
- 생산성 향상 
- 가격 인상 
- 위험 감소 
- 시장점유율 증가

---

### 시간 낭비 요인

| 유형 | 예시 |
|------|------|
| **내부적 요인** | 우선순위 미설정, 미루는 습관, 충동적 행동 |
| **외부적 요인** | 불필요한 회의, 잦은 방문객·전화, 예상치 못한 업무 |

---

### 시간계획 방법

1. 목표 설정(SMART)
2. 우선순위 결정(시간 매트릭스)
3. 소요시간 결정
4. 계획서 작성

#### SMART 법칙

| 알파벳 | 의미 | 설명 |
|--------|------|------|
| **S** | Specific (구체적) | 목표를 명확하고 구체적으로 설정 |
| **M** | Measurable (측정 가능) | 달성 여부를 수치로 확인 가능해야 함 |
| **A** | Action-oriented (행동 지향) | 구체적인 행동으로 실천할 수 있는 목표여야 함 |
| **R** | Realistic (현실적) | 자신의 상황·자원에 맞는 목표여야 함 |
| **T** | Time-bound (기한 설정) | 목표 달성 기한을 명확히 정해야 함 |

#### 시간 매트릭스 (우선순위 결정)

|  | **긴급함** | **긴급하지 않음** |
|---|---|---|
| **중요함** | 1사분면 — 즉시 처리 | 2사분면 — 계획적으로 처리 |
| **중요하지 않음** | 3사분면 — 위임 가능 | 4사분면 — 제거·최소화 |

> 시간관리의 핵심은 **2사분면(중요하지만 긴급하지 않은 일)에 충분한 시간을 투자**하는 것이다.

---

### 시간계획 수립 방법

#### 60:20:20 법칙

| 구분 | 비율 | 설명 |
|------|------|------|
| 계획된 행동 | **60%** | 예정된 업무·일정 |
| 자발적 행동 | **20%** | 창조성 발휘 |
| 계획 외의 행동 | **20%** | 돌발 상황 대비 여유 |

#### 그 외 시간계획 시 고려사항

① 어디에 어떻게 시간을 사용하고 있는지 행동과 시간, 시간관리 저해요인을 분석할 것
② 해당 시간에 예정된 일·행동을 모두 리스트화할 것
③ 시간계획을 규칙적·체계적으로 체크하여 일관성 있게 마무리할 것
④ 실현 가능한 것만 현실적으로 계획할 것
⑤ 시간계획이란 결국 목표달성을 위해 필요한 것임을 알고 유연성 있게 계획을 수립할 것
⑥ 시간의 손실이 일어났을 경우 미루지 않고 바로 보상할 것
⑦ 체크리스트나 스케줄 표 등을 사용하여 전체 상황을 파악할 것
⑧ 해야 할 일을 끝내지 못했을 경우 차기 계획에 반영할 것
⑨ 예정된 행동 외에 기대되는 성과와 행동의 목표도 기록할 것
⑩ 적절한 시간 프레임(Frame)을 설정하고 특정 일을 하는 데 필요한 시간만 계획에 포함할 것
⑪ 우선순위를 고려하여 여러 일 중에 어느 일을 먼저 처리할 것인지 결정할 것
⑫ 자신의 업무 일부를 부하에게 위임하고 그 수행 책임을 지도록 하는 권한위양을 실행할 것
> 권한위양은 조직을 탄력적으로 운용하고 조직구성원들의 근로의욕을 높여주는 데 효과적임

⑬ 예상치 못한 일들이 발생할 경우와 이동시간 및 대기시간과 같은 자유시간 등을 고려하여 여유시간을 확보할 것
⑭ 일의 경중에 따라 시간을 할애하며, 전체적인 계획을 정리하는 시간을 갖도록 할 것
⑮ 본인 이외의 사람들의 시간계획도 감안하여 계획을 수립할 것
