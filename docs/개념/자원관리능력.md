## 1. 자원의 개념 및 관리 과정

### 자원의 개념

자원이란 직업생활에서 목표를 달성하기 위해 활용되는 **시간, 돈(예산), 물적자원, 인적자원** 등을 의미한다.

| 자원 | 설명 |
|------|------|
| 시간 | 업무 수행에 사용 가능한 시간 |
| 예산 | 업무에 투입되는 비용·자금 |
| 물적자원 | 장비, 시설, 재료 등 물리적 자원 |
| 인적자원 | 업무를 수행하는 사람(인력, 역량) |

> 자원은 **유한하다** — 낭비 없이 효율적으로 활용해야 한다.

---

### 자원의 낭비요인

자원이 효과적으로 활용되지 못하는 주요 원인 4가지:

| 낭비요인 | 설명 |
|----------|------|
| **비계획적 행동** | 계획 없이 즉흥적으로 행동하여 자원을 낭비 |
| **편리성 추구** | 당장 편한 방법을 선택해 비효율 발생 |
| **자원에 대한 인식 부재** | 자원의 유한성을 인식하지 못해 무분별하게 사용 |
| **노하우 부족** | 효율적인 자원 활용 방법을 몰라 낭비 발생 |

---

### 효과적인 자원관리 과정

효과적인 자원관리는 아래 4단계 순서로 이루어진다:

```
① 필요한 자원 확인 → ② 이용 가능한 자원 수집 → ③ 자원 활용 계획 수립 → ④ 계획에 따른 수행
```

| 단계 | 내용 |
|------|------|
| **1단계: 필요한 자원 확인** | 업무 목표 달성에 어떤 자원이 얼마나 필요한지 파악 |
| **2단계: 이용 가능한 자원 수집** | 실제로 확보 가능한 자원의 종류·양을 조사 |
| **3단계: 자원 활용 계획 수립** | 수집된 자원을 어떻게 배분·사용할지 계획 작성 |
| **4단계: 계획에 따른 수행** | 계획대로 자원을 투입하고 실행 |

> **핵심 포인트:** 자원관리는 무작정 실행하는 것이 아니라, **계획 수립 후 수행**하는 것이 원칙이다.

## 2. 시간자원관리

### 시간의 특성

- 매일 24시간
- 속도가 일정
- 멈출 수 없음
- 빌리거나 저축할 수 없음
- 사용 방법에 따라 가치가 다름
- 시기에 따라 밀도가 다름

---

### 시간 관리의 효과

**개인 관점**
- 생산성 향상
- 스트레스 감소
- 균형적인 삶 
- 목표 성취

**기업 관점**
- 생산성 향상 
- 가격 인상 
- 위험 감소 
- 시장점유율 증가

---

### 시간 낭비 요인

| 유형 | 예시 |
|------|------|
| **내부적 요인** | 우선순위 미설정, 미루는 습관, 충동적 행동 |
| **외부적 요인** | 불필요한 회의, 잦은 방문객·전화, 예상치 못한 업무 |

---

### 시간계획 방법

1. 목표 설정(SMART)
2. 우선순위 결정(시간 매트릭스)
3. 소요시간 결정
4. 계획서 작성

#### SMART 법칙

| 알파벳 | 의미 | 설명 |
|--------|------|------|
| **S** | Specific (구체적) | 목표를 명확하고 구체적으로 설정 |
| **M** | Measurable (측정 가능) | 달성 여부를 수치로 확인 가능해야 함 |
| **A** | Action-oriented (행동 지향) | 구체적인 행동으로 실천할 수 있는 목표여야 함 |
| **R** | Realistic (현실적) | 자신의 상황·자원에 맞는 목표여야 함 |
| **T** | Time-bound (기한 설정) | 목표 달성 기한을 명확히 정해야 함 |

#### 시간 매트릭스 (우선순위 결정)

|  | **긴급함** | **긴급하지 않음** |
|---|---|---|
| **중요함** | 1사분면 — 즉시 처리 | 2사분면 — 계획적으로 처리 |
| **중요하지 않음** | 3사분면 — 위임 가능 | 4사분면 — 제거·최소화 |

> 시간관리의 핵심은 **2사분면(중요하지만 긴급하지 않은 일)에 충분한 시간을 투자**하는 것이다.

---

### 시간계획 수립 방법

#### 60:20:20 법칙

| 구분 | 비율 | 설명 |
|------|------|------|
| 계획된 행동 | **60%** | 예정된 업무·일정 |
| 자발적 행동 | **20%** | 창조성 발휘 |
| 계획 외의 행동 | **20%** | 돌발 상황 대비 여유 |

#### 그 외 시간계획 시 고려사항

① 어디에 어떻게 시간을 사용하고 있는지 행동과 시간, 시간관리 저해요인을 분석할 것
② 해당 시간에 예정된 일·행동을 모두 리스트화할 것
③ 시간계획을 규칙적·체계적으로 체크하여 일관성 있게 마무리할 것
④ 실현 가능한 것만 현실적으로 계획할 것
⑤ 시간계획이란 결국 목표달성을 위해 필요한 것임을 알고 유연성 있게 계획을 수립할 것
⑥ 시간의 손실이 일어났을 경우 미루지 않고 바로 보상할 것
⑦ 체크리스트나 스케줄 표 등을 사용하여 전체 상황을 파악할 것
⑧ 해야 할 일을 끝내지 못했을 경우 차기 계획에 반영할 것
⑨ 예정된 행동 외에 기대되는 성과와 행동의 목표도 기록할 것
⑩ 적절한 시간 프레임(Frame)을 설정하고 특정 일을 하는 데 필요한 시간만 계획에 포함할 것
⑪ 우선순위를 고려하여 여러 일 중에 어느 일을 먼저 처리할 것인지 결정할 것
⑫ 자신의 업무 일부를 부하에게 위임하고 그 수행 책임을 지도록 하는 권한위양을 실행할 것
> 권한위양은 조직을 탄력적으로 운용하고 조직구성원들의 근로의욕을 높여주는 데 효과적임

⑬ 예상치 못한 일들이 발생할 경우와 이동시간 및 대기시간과 같은 자유시간 등을 고려하여 여유시간을 확보할 것
⑭ 일의 경중에 따라 시간을 할애하며, 전체적인 계획을 정리하는 시간을 갖도록 할 것
⑮ 본인 이외의 사람들의 시간계획도 감안하여 계획을 수립할 것

## 3. 예산자원관리

### 예산의 정의

- **사전적 의미:** 필요한 비용을 미리 헤아려 계산함
- **넓은 의미:** 개인 및 조직의 수입과 지출에 관한 것도 포함

### 예산관리능력의 정의

이용 가능한 예산을 확인하고 어떻게 사용할 것인지 계획하여 계획대로 사용하는 능력.
최소의 비용으로 최대의 효과를 얻기 위해 필요함.

---

### 책정 비용과 실제 비용의 균형

| 비교 | 결과 |
|------|------|
| 책정 비용 **>** 실제 비용 | 경쟁력 손실 |
| 책정 비용 **<** 실제 비용 | 적자 발생 |
| 책정 비용 **=** 실제 비용 | 이상적 |

---

### 예산관리의 구성

예산관리에는 비용 산정, 예산 편성뿐만 아니라 예산을 통제하는 것까지 포함됨.

---

### 예산의 구성요소

| 비용 종류 | 설명 | 예시 |
|-----------|------|------|
| **직접비용** | 제품·서비스 창출에 직접 소비된 비용 | 재료비, 원료·장비, 시설, 인건비, 출장비 |
| **간접비용** | 생산에 간접적으로 소비된 비용 | 보험료, 건물관리비, 광고비, 통신비, 공과금 |

---

### 예산수립 방법

#### 예산수립의 절차

| 단계 | 내용 |
|------|------|
| **필요한 과업 및 활동 구명** | 예산 범위 내에서 수행해야 하는 활동과 소요예산을 정리 |
| **우선순위 결정** | 활동별 예산 지출 규모를 확인하고 우선순위를 확정 |
| **예산 배정** | 우선순위가 높은 활동부터 예산을 배정하고 사용 |

> - 확보 순서: 고정비 → 유동비 → 예비비
> - 삭감 순서: 유동비 → 예비비 → 고정비

#### 과업세부도 (WBS)

- 과제 및 활동계획 수립 시 가장 기본적인 수단으로 활용되는 그래프
- 모든 일을 중요한 범주에 따라 체계화해 구분하며, 구체성에 따라 2단계, 3단계, 4단계 등으로 구분 가능

| 1단계 | 2단계 | 3단계 | 4단계 |
|-------|-------|-------|-------|
| 생일파티 | 계획 | | |
| | 음식 준비 | | |
| | 각종 기구 | | |
| | 세팅 | 게임과 오락 | 배구 네트 설치 |
| | | | 수영장 청소 |
| | | 청소 | 집안 청소 |
| | | | 잔디 깎기 |
| | | 도구 | 테이블과 의자 설치 |
| | | | 텐트 설치 |
| | 파티 후 청소 | | |

#### 예산수립 시 주의사항

- 큰 단위의 예산을 수립할 경우 기관의 규정을 확인하여 반영해야 함

---

### 예산집행 관리

#### 예산집행 관리 방법

| 차원 | 방법 |
|------|------|
| **업무적 차원** | 예산집행 실적을 워크시트로 작성 |
| **개인적 차원** | 수기 또는 프로그램·앱을 활용하여 가계부 작성 |

#### 예산집행 시 주의사항

- 예산계획에 차질이 없도록 예산집행 내역과 계획을 지속적으로 비교·검토해야 함

## 4. 물적자원관리

### 물적자원의 개념

인간이 약한 신체적 특성을 보완하기 위하여 사용하는 자원으로, 세상에 존재하는 모든 물체가 포함됨.

### 물적자원의 종류

| 종류 | 설명 | 예시 |
|------|------|------|
| **자연자원** | 자연 상태에 있는 그대로의 자원 | 석유, 석탄, 나무 |
| **인공자원** | 인간이 인위적으로 가공하여 만든 자원 | 시설, 장비 |

### 물적자원관리의 중요성

- 효과적으로 관리할 경우 → 과제·사업의 성공으로 경쟁력 향상
- 관리가 부족할 경우 → 과제·사업 실패, 경제적 손실 발생

---

### 물적자원관리 절차

| 단계 | 내용 |
|------|------|
| **사용 물품과 보관 물품의 구분** | 활용 계획을 철저히 확인하여 꺼냈다가 다시 넣는 반복 작업을 방지하고 편리성을 높임 |
| **동일 및 유사 물품으로의 분류** | 동일성의 원칙(같은 품종은 같은 장소)·유사성의 원칙(유사품은 인접 장소)에 따라 보관 |
| **물품 특성에 맞는 보관 장소 선정** | 재질, 무게, 부피 등 물품 특성을 반영하여 보관 장소를 선정 |

### 회전대응 보관의 원칙

① 입·출하의 빈도가 높은 품목은 출입구에서 가까운 곳에 보관
② 물품의 활용 빈도가 상대적으로 높은 것은 가져다 쓰기 쉬운 위치에 보관

---

### 물적자원관리 방법

#### 바코드의 원리 활용

① 다량의 물품을 취급하는 곳에서는 바코드, QR코드 등을 활용하여 물품을 관리
② 개인의 사적인 물품도 바코드처럼 기호화하여 효과적으로 관리

#### 물품관리 프로그램 활용

① 쉽고 체계적으로 물품 관리를 할 수 있는 각종 프로그램이 개발되고 있음
② 개인보다는 기업·조직 차원에서 활용하는 경우가 많으며, 큰 조직에서 다량의 물품을 효과적으로 관리하는 데 도움이 됨

---

## 5. 인적자원관리

### 인적자원의 필요성

사업의 발달로 생산 현장이 첨단화·자동화되었으나, 물적자원과 예산 등의 생산요소를 효율적으로 결합시켜 가치를 창조하는 일은 사람이 하므로 기업 경영에서는 구성원의 자발적인 협력이 필요함.

### 인적자원관리의 개념

목적을 달성하기 위하여 필요한 인적자원을 조달·확보·유지·개발하여 경영 조직 내 구성원의 능력을 최고로 발휘하도록 하며, 근로자와 사용자 간의 협력 체계가 이루어지도록 관리하는 것.

### 효율적인 인적자원관리의 원칙

① 적재적소 배치의 원칙
② 공정 보상의 원칙
③ 공정 인사의 원칙
④ 종업원 안정의 원칙
⑤ 창의력 계발의 원칙
⑥ 단결의 원칙

---

### 인적자원관리의 중요성

#### 기업적 차원

조직의 성과는 인적자원에 대한 관리의 영향을 크게 받으며, 이는 인적자원의 특성에서 비롯됨.

| 특성 | 설명 |
|------|------|
| **능동성** | 예산·물적자원과 달리 능동적·반응적 성격을 지님. 성과는 욕구·동기·태도·만족감에 따라 달라지며 경영관리에 의해 조건화됨 |
| **개발가능성** | 자연적 성장과 성숙은 물론, 오랜 기간에 걸쳐 개발될 수 있는 잠재능력과 자질을 보유함 |
| **전략적 자원** | 다양한 자원을 활용하는 것은 결국 사람이므로 다른 어떤 자원보다도 전략적 중요성이 강조됨 |

#### 개인적 차원

① 각종 정보와 정보의 소스를 획득할 수 있음
② 나 자신의 인간관계와 행동에 대해서 알 수 있으며, 삶의 탄력을 얻을 수 있음
③ 참신한 아이디어나 도움을 얻을 수 있음

### 개인의 인적자원관리 (= 인맥관리)

- 인맥: 가족, 친구, 직장동료 등 자신이 알고 있거나 관계를 형성하고 있는 다양한 사람들

| 구분 | 설명 |
|------|------|
| **핵심 인맥** | 자신과 직접적인 관계가 있는 사람들 |
| **파생 인맥** | 핵심 인맥을 통해 알게 된 사람, 우연히 알게 된 사람 등 |

#### 인맥관리 방법

**① 명함관리**
- 받은 명함을 인맥관리에 적극적으로 사용
- 명함관리 프로그램·애플리케이션 활용
- 명함에 상대에 대한 구체적인 정보를 메모

**② 인맥관리카드 작성**
- 이름, 관계, 직장·부서, 학력, 출신지, 연락처, 친한 정도 등을 정리하여 관리
- 핵심·파생 인맥을 구분하여 작성하며, 파생 인맥 카드에는 어떤 관계에 의해 파생되었는지 포함

**③ 소셜네트워크(SNS, Social Network Service)**
- 시간과 공간을 초월하여 네트워크상에서 인맥을 형성·관리
- 비즈니스 특화: BNS(Business social Network Service) — 예) 링크드인(LinkedIn)

### 팀원 관리 방법

#### 인력 배치의 원칙

| 원칙 | 설명 |
|------|------|
| **적재적소주의** (The right man for the right job) | 팀원의 능력·성격과 가장 적합한 위치에 배치하여 개개인의 능력을 최대로 발휘하도록 함 |
| **능력주의** | 능력을 발휘할 기회와 장소를 부여하고, 성과를 바르게 평가하며 능력·실적에 상응하는 보상을 줌 |
| **균형주의** | 전체와 개체가 균형을 이룰 수 있도록 모든 팀원에 대해 평등한 적재적소를 고려함 |

#### 인력 배치의 유형

| 유형 | 설명 |
|------|------|
| **양적 배치** | 작업량과 조업도, 여유·부족 인원을 감안하여 소요 인원을 결정·배치 |
| **질적 배치** (= 적재적소의 배치) | 능력이나 성격 등과 가장 적합한 위치에 배치 |
| **적성 배치** | 팀원의 적성 및 흥미에 따라 배치 |

> 양적·질적·적성 배치는 따로 분리되는 것이 아닌 적절히 조화하여 운영되어야 함

#### 과업세부도

인력을 배치하여 업무를 수행하는 과정에서 팀원들에게 할당된 일을 적절히 관리하기 위해서는 과업세부도를 작성하는 것이 효과적임.
