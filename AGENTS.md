# AGENTS.md

## 프로젝트 목적

NCS(국가직무능력표준) 복습용 저장소.

- 모르는 개념을 정리하고 질문하는 공간
- 오답 체크 및 틀린 내용 기록
- Codex에게 질문하고 피드백 받는 용도

## 기록 규칙

- 오답은 파일에 직접 기록 (왜 틀렸는지 이유 포함)
- 개념 정리는 과목/모듈별로 파일 분리
- 본문 내용이나 제목이 바뀌면 해당 문서에 목차가 있는 경우 목차도 함께 수정

## 문제 파일 형식

`docs/Codex 생성 문제/` 안의 모든 문제 파일은 아래 형식을 따름

```markdown
---

**N.** 문제 내용

① 보기1  
② 보기2  
③ 보기3  
④ 보기4  
⑤ 보기5  

<details>
<summary>정답 및 해설</summary>

**정답: X**

해설 내용

</details>

출처: ## 섹션명 > ### 서브섹션명
```

- 정답과 해설은 `<details><summary>` 태그로 접어서 표시
- 각 문제 끝에 `출처:` 로 원본 섹션(`##` > `###`) 기록
- 보기 줄 끝에 `  `(공백 2칸) 붙여 줄바꿈 처리
