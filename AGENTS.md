# AGENTS.md

## 프로젝트 목적

NCS(국가직무능력표준) 복습용 저장소.

- 모르는 개념을 정리하고 질문하는 공간
- 오답 체크 및 틀린 내용 기록
- Codex에게 질문하고 피드백 받는 용도

## 기록 규칙

- 오답은 파일에 직접 기록 (왜 틀렸는지 이유 포함)
- 개념 정리는 과목/모듈별로 파일 분리
- 본문 내용이나 제목이 바뀌면 해당 문서에 목차가 있는 경우 목차도 함께 수정
- 목록은 깊이와 상관없이 같은 레벨 항목끼리 빈 줄을 넣고, 상위 항목과 첫 하위 항목은 바로 아래에 붙여 씀
- 기존 목록이나 섹션에 내용을 옮기거나 추가할 때는 항목 배열이 의미순·절차순·중요도순 등으로 정렬되어 있는지 먼저 확인하고, 정렬 의도가 보이면 문맥에 맞는 위치에 삽입
- 순서가 중요한 절차는 번호 목록을 사용하고, 단순 유형·분류·예시는 불렛 목록을 우선 사용
- 한글순 정렬 요청 시 항목 첫 글자의 초성을 `ㄱ ㄴ ㄷ ㄹ ㅁ ㅂ ㅅ ㅇ ㅈ ㅊ ㅋ ㅌ ㅍ ㅎ` 순서로 검산한 뒤 적용
- 문제가 한 번 발생하면 원인을 확인하고, 같은 문제가 재발하지 않도록 작업 방식이나 규칙을 스스로 개선

## 문제 파일 형식

`docs/Codex 생성 문제/` 안의 모든 문제 파일은 아래 형식을 따름

```markdown
---

**N.** 문제 내용

① 보기1  
② 보기2  
③ 보기3  
④ 보기4  
⑤ 보기5  

<details>
<summary>정답 및 해설</summary>

**정답: X**

해설 내용

</details>

출처: ## 섹션명 > ### 서브섹션명
```

- 정답과 해설은 `<details><summary>` 태그로 접어서 표시
- 각 문제 끝에 `출처:` 로 원본 섹션(`##` > `###`) 기록
- 보기 줄 끝에 `  `(공백 2칸) 붙여 줄바꿈 처리
